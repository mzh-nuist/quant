## 研究模块

### 模块 1：驱动力归因（复用 Q1）

- CSI 300 PE/PB 双重分解（数据已有）
- CSI 300 估值分位（PE/PB 滚动 10 年）
- 拥挤度 + 相关性集中度
- 风格周期检测 → CSI 300 / SSE 50 比值 + 成长/价值比值

**产出**：CSI 300 当前处于什么位置？估值驱动还是盈利驱动？

### 模块 2：因子截面检验（复用 Q3 七层验证，V1 核心）

**因子集**：

| 类别 | 因子 | 计算方式 |
|------|------|---------|
| 估值 | PB、PE | baostock 财报 or akshare |
| 质量 | ROE、ROIC | baostock `query_profit_data` |
| 成长 | 营收增速、净利增速 | baostock `query_growth_data` |
| 低波动 | `vol_2025`（2025 年日 K 年化波动率） | 日 K 对数收益率 std × √252 |
| 动量 | `ret_2025`（2025 全年收益率） | 日 K 首尾价格比 |
| **新增** | 股息率 | akshare or baostock |
| **新增** | 北向持股变化 | akshare `stock_hsgt_hold_em` |
| 情绪 | 换手率变化 | `turnover_2026/avg_turnover_2025 - 1` |

**验证层次**：

```
t 检验 → Mann-Whitney U → Logit 回归 → 五分位单调性
→ 行业 × 因子矩阵 → 市值分层（500 亿/1000 亿/2000 亿）
```

**被解释变量**：`resilient` = 2026 YTD > 0（与 Q3 一致，保证可比），同时增加**超额收益版本**（YTD > CSI 300 指数涨幅）。

### 模块 3：大盘独有维度

- 北向资金净流入 vs 未来收益
- 股息率——能否替代 PB 作为大盘估值锚？
- 机构抱团股——持仓集中度与抗跌关系

### 模块 4：跨市值对比（Q5 vs Q3）

同一因子在微盘 vs 大盘的方向对比：
- 方向一致 → 跨市值稳健因子
- 方向相反 → 市值套利机会
- 只有一端正交 → 纯结构效应

## 关键假设（事后跑数据验证）

1. 质量因子方向反转：高 ROE/高 EPS → 抗跌
2. PB 方向反转：低 PB → 超额收益
3. 波动率方向一致：低波 → 抗跌（跨市值最稳健）
4. 北向持续流入 > 1 月 → 抗跌信号
5. 股息率独立于 PB 提供增量信息

### 可能被推翻的

- 质量因子在不显著 → A 股整体定价效率低，不限于微盘
- PB 仍然是陷阱 → PB-ROE 框架在 A 股整体失效
- 动量仍然 50% 随机 → 动量根本不适合 A 股，与市值无关
