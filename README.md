## 策略原理

### 什么是双均线交叉？

双均线交叉（SMA Crossover）是最经典的趋势跟踪策略之一，核心逻辑非常简单：

| 信号 | 条件 | 含义 |
|------|------|------|
| **金叉** (Golden Cross) | 短期均线 **上穿** 长期均线 | 短期趋势转强，看涨 → **买入** |
| **死叉** (Death Cross) | 短期均线 **下穿** 长期均线 | 短期趋势转弱，看跌 → **卖出** |

```
        金叉 ↑                         死叉 ↓
          \  短期均线                     \  长期均线
     ──────\──── 长期均线          ────────\──── 短期均线
            \                              \
     ────────\─────────            ──────────\───────
              \                              \
```

### 数学公式

简单移动平均（SMA）的计算公式：

$$
\text{SMA}_{n}(t) = \frac{1}{n} \sum_{i=0}^{n-1} P_{t-i}
$$

其中 $P_t$ 为第 $t$ 日的收盘价，$n$ 为均线周期。

### 为什么均线交叉有效？

1. **趋势跟踪**：均线天然具有平滑价格波动的特性，能过滤市场噪音。
2. **惯性原理**：市场趋势一旦形成往往具有一定的持续性。
3. **心理锚定**：均线被大量交易者关注，本身具有自我实现的特性。

### 策略局限性

- 在**震荡市**中，均线频繁交叉会产生大量虚假信号，导致连续亏损。
- 均线是**滞后指标**，金叉/死叉发生时趋势往往已经走出一段。
- 单边急涨急跌行情中表现较好，但横盘整理行情中表现较差。

## 快速开始

### 默认运行（贵州茅台 600519，10/30 均线）

```bash
conda activate quant
python main.py
```

程序会自动：
1. 通过 akshare 拉取贵州茅台 2020~2025 年的前复权日线数据
2. 使用 10 日均线和 30 日均线运行交叉策略回测
3. 在控制台输出每笔交易日志和最终收益率
4. 弹窗显示 K 线图 + 成交量 + 两条均线的可视化图表

### 自定义股票和参数

```bash
# 平安银行 000001，5 日与 20 日均线交叉
python main.py -c 000001 -s 5 -l 20

# 比亚迪 002594，20 日与 60 日均线，50 万初始资金
python main.py -c 002594 -s 20 -l 60 --cash 500000

# 指定回测日期范围
python main.py -c 600519 --start 20180101 --end 20231231

# 仅输出文字结果，不显示图表
python main.py -c 600519 --no-plot
```

## 代码说明

### `data_fetcher.py` — 数据获取

- 使用 `akshare.stock_zh_a_hist()` 获取 A 股历史日线数据。
- **自动缓存**：首次获取后存为 CSV，下次运行直接从本地加载，大幅提升启动速度。
- **前复权处理**：默认使用 `qfq`（前复权），保证价格连续性，适合回测。
- **列名标准化**：将 akshare 的中文列名映射为 backtrader 所需的英文标准列名。

### `sma_crossover.py` — 策略核心

- 继承 `bt.Strategy`，通过 `params` 定义可配置的均线周期。
- 使用 `bt.indicators.SimpleMovingAverage` 计算短/长期均线。
- 使用 `bt.indicators.CrossOver` 判断交叉方向（+1 为上穿，-1 为下穿）。
- `next()` 方法在每个 bar 检查信号并执行买卖。
- `notify_order()` / `notify_trade()` 记录每笔订单和交易的详细信息。
- `stop()` 方法输出回测汇总统计。

### `main.py` — 主程序

- `configure_matplotlib()` 自动检测系统中文字体，配置 seaborn 风格图表的全局样式。
- 使用 `argparse` 提供灵活的命令行参数。
- `run_backtest()` 串联数据获取 → Cerebro 初始化 → 策略添加 → 回测执行 → 结果输出整个流程。
- 图表使用 backtrader 内置 `cerebro.plot()` 绘制，包含 K 线、成交量柱和均线叠加。

## 参数优化建议

| 市场风格 | 推荐短周期 | 推荐长周期 | 说明 |
|----------|-----------|-----------|------|
| 短线交易 | 5 | 20 | 对趋势变化敏感，信号多，手续费影响大 |
| 中线波段 | 10 | 30 | 平衡型，过滤部分噪音但不过于滞后（默认） |
| 长线趋势 | 20 | 60 | 信号少但可靠性高，适合大资金低频交易 |
| 超长趋势 | 50 | 200 | 著名"黄金交叉/死亡交叉"，信号极少 |

> **提示**：最佳的均线参数因股票和时间段而异，建议对不同参数组合进行系统化回测比较，找到适合目标标的的最优参数。切勿仅凭单次回测结果进行实盘交易。

## 常见问题

### Q1: 运行报错 "未找到中文字体"

图表中文标题和标签可能显示为方框。解决方法：
- **Windows**：通常已自带 SimHei（黑体），正常不会出现此问题。
- **macOS**：系统自带 PingFang SC，一般正常。
- **Linux**：安装中文字体 `sudo apt install fonts-noto-cjk`。

### Q2: 数据获取失败

- 检查网络连接是否正常。
- akshare 数据源偶尔会有维护，稍后重试即可。
- 确认股票代码是否正确（上证 6 开头，深证 0/3 开头）。

### Q3: 回测收益率与实盘不符

回测与实际交易存在以下差异：
- **滑点**：回测按收盘价成交，实盘存在买卖价差和滑点。
- **流动性**：回测假设订单全部成交，实盘可能无法在预期价格成交。
- **市场冲击**：大资金交易会影响市场价格，回测未考虑。
- **幸存者偏差**：历史数据可能不包含已退市股票。

### Q4: 如何添加更多技术指标？

在 `sma_crossover.py` 的 `__init__` 中添加 backtrader 内置指标即可：

```python
# 示例：添加 RSI 和 MACD
self.rsi = bt.indicators.RSI(self.data.close, period=14)
self.macd = bt.indicators.MACD(self.data.close)
```
